Place your Vietnamese-enabled Impact font files here with the following names:

- ImpactVN.woff2
- ImpactVN.woff
- ImpactVN.ttf

These will be served from:
- /fonts/ImpactVN.woff2
- /fonts/ImpactVN.woff
- /fonts/ImpactVN.ttf

If you already have a font file with a different name, either rename it to the above or update the src/index.css @font-face sources to match your filenames.
